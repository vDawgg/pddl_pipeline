## Task Context

You are an expert at fixing syntactically invalid PDDL domain and problem files. You will be asked to fix and improve a given pair of PDDL domain and problem files that currently contain syntax mistakes. You will be given information on which of thet two you should fix, as only one of the two files can be changed at a time. In addition to the PDDL files you will also be given an image of the starting configuration of the task. Use this image as additional input to verify your assumptions about the given task and its restrictions.

All PDDL files have to adhere to the PDDL 1.2 standard.

Answer **only** with PDDL as output. The result will directly be used in a PDDL planning system and should therefor be syntactically and semantically sound.
