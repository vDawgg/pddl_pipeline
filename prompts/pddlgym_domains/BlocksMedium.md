## Domain description

A robot is tasked to re-assemble stackable blocks on a table with unlimited space. The robot can be used for stacking a block onto a block, unstacking a block from a block, putting down a block, or picking up a block.

The available actions are:
- pick-up(x - block)
- put-down(x - block)
- stack(x - block, y - block)
- unstack(x - block, y - block)
