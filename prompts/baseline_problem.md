## Task description

Your task is to generate the PDDL problem given the following problem description and the previously generated PDDL domain.

There are 5 given blocks on the table. Blocks A is on the table and block B is on block A. Blocks C, D and E are all on the table and accessible by the robot arm. The blocks should be reoriented such that Block B is on the table and the other blocks are staccked on top of B in the following order: C, D, E, A.
